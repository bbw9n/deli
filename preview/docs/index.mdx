# Deli Overview

`deli` is the terminal console for end-to-end control-plane work.

<Info>
Use deli when you want docs, config, code, and monitoring in one place.
</Info>

## Core workflow

- Read system and runbook documentation
- Inspect structured runtime configuration
- Jump into the relevant worktree or command
- Review metrics before and after an operation

## Views

### Documents

Browse Mintlify-style docs and markdown without leaving the terminal.

### Configs

Inspect provider-backed configuration as typed tabular data.

### Worktree

Use command launchers and workspace shortcuts as operational jump points.

### Monitoring

Review time-series charts with terminal-native fallbacks.
